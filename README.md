## Getting Started


To get a local copy up and running follow these simple example steps.

### Prerequisites
  - Git installed

### Setup
  - open a terminal
  - run ``` git clone https://github.com/Leboroz/Hello-Microverse.git```
  - run ``` cd Hello-Microverse/views```
  - open index.html
